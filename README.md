# ☕ Coffee Journal

咖啡爱好者的本地记录工具，打开浏览器即用，数据存在本地。

## 功能特性

### 🫘 咖啡豆库
- 记录咖啡豆完整信息：豆名、产地、品种、处理法、烘焙度、海拔、烘焙商
- 烘焙度可视化展示（SCA 标准 7 级，含 Agtron 色值和温度范围）
- 风味标签选择（基于 SCA 风味轮，12 大类 72 种细分风味）
- 购买信息记录：价格、重量、渠道
- 豆子状态管理：未开袋 / 饮用中 / 已喝完
- 搜索功能：按豆名、产地、烘焙商搜索

### 📝 冲煮记录
- 支持 10 种冲煮方式：手冲、冰手冲、意式浓缩、法压壶、摩卡壶、爱乐压、冷萃、虹吸壶、Chemex、其他
- 完整参数记录：粉量、水量、水温、时间、研磨度、磨豆机
- 自动计算粉水比
- 选择冲煮方式后自动填充推荐参数
- 口感评分：总分 + 香气 / 酸质 / 醇厚 / 甜感四维度（1-10 分）
- 风味笔记与备注

### 📊 数据分析
- 总览：冲煮次数、咖啡豆种类、平均评分、平均粉量
- 冲煮方式分布柱状图
- 最爱咖啡豆 TOP5 排行
- 评分趋势图（近 20 次）
- 风味词云（综合豆子标签和冲煮风味笔记）
- 月度冲煮量统计

### ⚙️ 设置
- 默认冲煮参数配置（粉量、水温）
- 数据导出 / 导入（JSON 格式）
- 清空所有数据

## 技术栈

- **前端**: HTML + CSS + 原生 JavaScript
- **数据存储**: 浏览器 localStorage
- **无后端依赖**，打开 `index.html` 即可使用

## 项目结构

```
coffee-journal/
├── index.html      # 页面结构与模态框
├── style.css       # 样式（响应式，支持移动端）
├── app.js          # 应用逻辑（数据层 + 渲染 + CRUD）
└── README.md
```

## 快速开始

直接在浏览器中打开 `index.html` 即可使用，无需安装任何依赖。

```bash
git clone https://github.com/liuxing-fly/coffee-journal.git
cd coffee-journal
open index.html
```

建议使用"导出 JSON"功能定期备份数据。

## 数据存储

所有数据存在浏览器 localStorage 中，键名：
- `cj_beans` - 咖啡豆列表
- `cj_brews` - 冲煮记录列表
- `cj_settings` - 用户设置

数据不会上传到任何服务器，清除浏览器数据会导致丢失，请定期导出备份。

## 烘焙度参考（SCA 标准）

| 级别 | 中文 | Agtron | 风味特征 |
|------|------|--------|----------|
| 1 | 极浅烘 | 90-100 | 保留产地风味，花香果酸极致 |
| 2 | 浅烘 | 80-95 | 保留原始风味，酸度明亮 |
| 3 | 浅中烘 | 65-80 | 平衡酸甜，花果香明显 |
| 4 | 中烘 | 55-65 | 均衡口感，坚果巧克力调 |
| 5 | 中深烘 | 45-55 | 焦糖甜感，醇厚度增加 |
| 6 | 深烘 | 35-45 | 浓郁烟熏，苦甜交织 |
| 7 | 极深烘 | 25-35 | 强烈焦苦，油脂丰富 |
